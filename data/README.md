## Dataset Information

### Source

This project uses the **Retail Store Product Sales Dataset** obtained from Kaggle.

**Dataset Link:**  
https://www.kaggle.com/datasets/ranaghulamnabi/retail-store-product-sales-dataset-analysis

### Description

The dataset contains retail sales and operational variables used to analyze customer purchasing behavior, operational performance, and return rate risk.

#### Variables

- Price
- Discount
- Promotion Intensity
- Footfall
- Ad Spend
- Competitor Price
- Stock Level
- Weather Index
- Customer Sentiment
- Return Rate

### Usage

This dataset was used for:

- Data Cleaning (Excel)
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Multiple Linear Regression
- Two-Way ANOVA
- Dashboard Development (Power BI)

###Note

The dataset is publicly available on Kaggle. If it is not included in this repository, please download it directly from the original source.
